GoиҜӯиЁҖз»“жһ„зұ»еһӢиҜҰи§Ј

Goе…Ғи®ёз”ЁжҲ·иҮӘе®ҡд№үзұ»еһӢпјҢеҪ“дҪ йңҖиҰҒз”Ёд»Јз ҒжҠҪиұЎжҸҸиҝ°дёҖдёӘдәӢзү©жҲ–иҖ…еҜ№иұЎзҡ„ж—¶еҖҷпјҢеҸҜд»ҘеЈ°жҳҺдёҖдёӘ struct зұ»еһӢжқҘиҝӣиЎҢжҸҸиҝ°гҖӮ

еҪ“з„¶пјҢGoиҜӯиЁҖдёӯпјҢз”ЁжҲ·иҝҳеҸҜд»ҘеҹәдәҺе·Іжңүзҡ„зұ»еһӢжқҘе®ҡд№үе…¶д»–зұ»еһӢгҖӮ

з®ҖеҚ•жқҘиҜҙпјҢGoиҜӯиЁҖдёӯз”ЁжҲ·еҸҜд»ҘжңүдёӨз§Қж–№жі•е®ҡд№үзұ»еһӢпјҢз¬¬дёҖз§ҚжҳҜдҪҝз”Ё struct е…ій”®еӯ—жқҘеҲӣйҖ дёҖдёӘз»“жһ„зұ»еһӢпјӣз¬¬дәҢз§ҚжҳҜеҹәдәҺе·Іжңүзҡ„зұ»еһӢпјҢе°Ҷе…¶дҪңдёәж–°зұ»еһӢзҡ„зұ»еһӢиҜҙжҳҺгҖӮ

01.В иҮӘе®ҡд№үзұ»еһӢзҡ„еҹәжң¬дҪҝз”Ё

еҹәдәҺе·Іжңүзҡ„зұ»еһӢзҡ„иҝҷз§Қж–№ејҸжҜ”иҫғз®ҖеҚ•пјҢдҪҶйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢиҷҪз„¶жҳҜеҹәдәҺе·Іжңүзұ»еһӢжқҘе®ҡд№үж–°зұ»еһӢпјҢдҪҶжҳҜеҹәзЎҖзұ»еһӢе’Ңж–°зұ»еһӢжҳҜе®Ңе…ЁдёҚеҗҢзҡ„дёӨз§Қзұ»еһӢпјҢдёҚиғҪзӣёдә’иөӢеҖјпјҢеӣ дёәGoиҜӯиЁҖдёӯпјҢзј–иҜ‘еҷЁдёҚдјҡеҜ№дёҚеҗҢзұ»еһӢзҡ„еҖјеҒҡйҡҗејҸиҪ¬жҚўгҖӮ

еҪ“йңҖиҰҒдҪҝз”ЁдёҖдёӘжҜ”иҫғжҳҺзЎ®зҡ„еҗҚеӯ—зұ»жҸҸиҝ°дёҖз§Қзұ»еһӢж—¶пјҢдҪҝз”Ёиҝҷз§ҚиҮӘе®ҡд№үзұ»еһӢе°ұжҜ”иҫғеҗҲйҖӮпјҢжҜ”еҰӮе®ҡд№үдёҖдёӘиЎЁзӨәе№ҙйҫ„зҡ„зұ»еһӢеҸҜд»ҘеҹәдәҺж•ҙеҪўжқҘе®ҡд№үдёҖдёӘ Age зұ»еһӢпјҢзү№жҢҮе№ҙйҫ„зұ»еһӢгҖӮ

дёӢйқўжҳҜеҹәдәҺе·Іжңүзұ»еһӢзҡ„ж–№ејҸе®ҡд№үзұ»еһӢзҡ„зӨәдҫӢ

дҪҝз”Ёе…ій”®еӯ— struct жқҘеЈ°жҳҺдёҖдёӘз»“жһ„зұ»еһӢж—¶пјҢиҰҒжұӮеӯ—ж®өжҳҜеӣәе®ҡе№¶дё”е”ҜдёҖзҡ„пјҢе№¶дё”еӯ—ж®өзҡ„зұ»еһӢд№ҹжҳҜе·ІзҹҘзҡ„пјҢдҪҶжҳҜеӯ—ж®өзұ»еһӢеҸҜд»ҘжҳҜеҶ…зҪ®зұ»еһӢпјҲжҜ”еҰӮ string, bool, int зӯүзӯүпјүпјҢд№ҹеҸҜд»ҘжҳҜз”ЁжҲ·иҮӘе®ҡд№үзҡ„зұ»еһӢпјҲжҜ”еҰӮпјҢжң¬ж–Үдёӯд»Ӣз»Қзҡ„ struct зұ»еһӢпјүгҖӮ

еЈ°жҳҺstruct з»“жһ„дҪ“зҡ„е…¬ејҸпјҡtype з»“жһ„дҪ“еҗҚз§° struct {}В гҖӮ

еңЁд»»дҪ•ж—¶еҖҷпјҢеҲӣе»әдёҖдёӘеҸҳйҮҸе№¶еҲқе§ӢеҢ–е…¶йӣ¶еҖјж—¶пјҢжҲ‘д»¬д№ жғҜжҳҜдҪҝз”Ёе…ій”®еӯ— varпјҢиҝҷз§Қз”Ёжі•жҳҜдёәдәҶжӣҙжҳҺзЎ®зҡ„иЎЁзӨәеҸҳйҮҸиў«и®ҫзҪ®дёәйӣ¶еҖјгҖӮ

иҖҢеҰӮжһңжҳҜеҸҳйҮҸиў«еҲқе§ӢеҢ–дёәйқһйӣ¶еҖјж—¶пјҢеҲҷдҪҝз”ЁзҹӯеҸҳйҮҸж“ҚдҪңз¬ҰВ :=В е’Ңз»“жһ„еӯ—йқўйҮҸ з»“жһ„зұ»еһӢ{ еӯ—ж®ө: еӯ—ж®өеҖј, } жҲ–иҖ… з»“жһ„зұ»еһӢ{ еӯ—ж®ө1еҖј, еӯ—ж®ө2еҖј } жқҘеҲӣе»әеҸҳйҮҸгҖӮ

дёӨз§Қеӯ—йқўйҮҸеҲқе§ӢеҢ–ж–№ејҸзҡ„е·®ејӮдёҺйҷҗеҲ¶пјҡ

з»“жһ„зұ»еһӢ{ еӯ—ж®ө1еҖј, еӯ—ж®ө2еҖј } иҝҷз§ҚеҲқе§ӢеҢ–ж–№ејҸж—¶пјҡ

  1. еңЁжңҖеҗҺдёҖдёӘеӯ—ж®өеҖјзҡ„з»“е°ҫеҸҜд»ҘдёҚз”ЁеҠ йҖ—еҸ· ,

  2. еҝ…йЎ»дёҘж јжҢүз…§еЈ°жҳҺж—¶зҡ„еӯ—ж®өйЎәеәҸжқҘиҝӣиЎҢеҲқе§ӢеҢ–пјҢдёҚз„¶дјҡеҫ—дёҚеҲ°йў„жңҹзҡ„з»“жһңпјӣеҰӮжһңеӯ—ж®өзұ»еһӢдёҚдёҖиҮҙпјҢиҝҳдјҡеҜјиҮҙеҲқе§ӢеҢ–еӨұиҙҘ

  3. еҝ…йЎ»иҰҒеҲқе§ӢеҢ–жүҖжңүзҡ„еӯ—ж®өпјҢдёҚз„¶дјҡжҠҘй”ҷ Too few values

з»“жһ„зұ»еһӢ{ еӯ—ж®ө: еӯ—ж®өеҖј, } иҝҷз§ҚеҲқе§ӢеҢ–ж–№ејҸж—¶пјҡ

  1. жҜҸдёҖдёӘеӯ—ж®өеҖјзҡ„з»“е°ҫеҝ…йЎ»иҰҒеҠ дёҖдёӘйҖ—еҸ· ,

  2. еҲқе§ӢеҢ–ж—¶пјҢдёҚиҰҒиҖғиҷ‘еӯ—ж®өеЈ°жҳҺзҡ„йЎәеәҸ

  3. е…Ғи®ёеҸӘеҲқе§ӢеҢ–йғЁеҲҶеӯ—ж®ө

д»ҘдёҠжҳҜ struct з»“жһ„зұ»еһӢзҡ„еҹәжң¬дҪҝз”ЁпјҢдҪҶжҳҜеңЁйЎ№зӣ®ејҖеҸ‘дёӯдјҡйҒҮеҲ°е…¶д»–зҡ„з”Ёжі•пјҢжҜ”еҰӮи§Јжһҗ json жҲ–иҖ… xml ж–Үд»¶еҲ°з»“жһ„дҪ“зұ»еһӢеҸҳйҮҸдёӯгҖӮ

02.В е…¬ејҖжҲ–жңӘе…¬ејҖзҡ„ж ҮиҜҶз¬Ұ

еңЁGoиҜӯиЁҖдёӯпјҢеЈ°жҳҺзұ»еһӢгҖҒеҮҪж•°гҖҒж–№жі•гҖҒеҸҳйҮҸзӯүж ҮиҜҶз¬Ұж—¶пјҢдҪҝз”ЁеӨ§е°ҸеҶҷеӯ—жҜҚејҖеӨҙжқҘеҢәеҲҶиҜҘж ҮиҜҶз¬ҰжҳҜеҗҰе…¬ејҖпјҲеҚіжҳҜеҗҰиғҪеңЁеҢ…еӨ–и®ҝй—®пјүгҖӮ

еӨ§еҶҷеӯ—жҜҚејҖеӨҙиЎЁзӨәе…¬ејҖпјҢе°ҸеҶҷеӯ—жҜҚејҖеӨҙиЎЁзӨәйқһе…¬ејҖгҖӮжүҖд»ҘеҰӮжһңжҹҗдёӘз»“жһ„зұ»еһӢд»ҘеҸҠз»“жһ„зұ»еһӢзҡ„еӯ—ж®өпјҢеҮҪж•°пјҢж–№жі•пјҢеҸҳйҮҸзӯүж ҮиҜҶз¬ҰпјҢжғіиҰҒиў«еӨ–йғЁи®ҝй—®еҲ°пјҢйӮЈеҝ…йЎ»д»ҘеӨ§еҶҷеӯ—жҜҚејҖеӨҙгҖӮ

03.В з»ҷиҮӘе®ҡд№үзұ»еһӢеўһеҠ ж–№жі•

еңЁGoиҜӯиЁҖдёӯпјҢзј–иҜ‘еҷЁеҸӘе…Ғи®ёдёәе‘ҪеҗҚзҡ„з”ЁжҲ·е®ҡд№үзҡ„зұ»еһӢеЈ°жҳҺж–№жі•гҖӮж–№жі•и·ҹеҮҪж•°зұ»дјјпјҢеҸӘжҳҜж–№жі•дёҚдјҡеҚ•зӢ¬еӯҳеңЁпјҢдёҖиҲ¬жҳҜз»‘е®ҡеҲ°жҹҗдёӘз»“жһ„зұ»еһӢдёӯпјҢз»ҷзұ»еһӢеўһеҠ ж–№жі•зҡ„ж–№ејҸеҫҲз®ҖеҚ•пјҢе°ұжҳҜеңЁж–№жі•еҗҚе’Ң func д№Ӣй—ҙеўһеҠ дёҖдёӘеҸӮж•°еҚіеҸҜ, иҝҷдёӘеҸӮж•°з§°дёәж–№жі•зҡ„жҺҘж”¶иҖ…гҖӮ

ж–№жі•зҡ„жҺҘж”¶иҖ…пјҢеҸҜд»ҘжҳҜеҖјжҺҘж”¶иҖ…пјҢд№ҹеҸҜд»ҘжҳҜжҢҮй’ҲжҺҘж”¶иҖ…гҖӮ

иҖҢеә”иҜҘдҪҝз”ЁеҖјжҺҘж”¶иҖ…иҝҳжҳҜжҢҮй’ҲжҺҘж”¶иҖ…пјҢйӮЈиҰҒзңӢз»ҷиҝҷдёӘзұ»еһӢеўһеҠ жҲ–еҲ йҷӨжҹҗдёӘеҖјж—¶пјҢжҳҜеҲӣе»әдёҖдёӘж–°еҖјпјҢиҝҳжҳҜиҰҒжӣҙж”№еҪ“еүҚеҖјпјҹеҰӮжһңжҳҜиҰҒеҲӣе»әдёҖдёӘж–°еҖјпјҢиҜҘзұ»еһӢзҡ„ж–№жі•е°ұдҪҝз”ЁеҖјжҺҘж”¶иҖ…пјӣеҰӮжһңжҳҜиҰҒдҝ®ж”№еҪ“еүҚеҖјпјҢе°ұдҪҝз”ЁжҢҮй’ҲжҺҘж”¶иҖ…гҖӮ

04.В еөҢе…Ҙзұ»еһӢ

GoиҜӯиЁҖйҖҡиҝҮзұ»еһӢеөҢеҘ—зҡ„ж–№ејҸжқҘеӨҚз”Ёд»Јз ҒпјҢеҪ“еӨҡдёӘз»“жһ„зұ»еһӢзӣёдә’еөҢеҘ—ж—¶пјҢеӨ–йғЁзұ»еһӢдјҡеӨҚз”ЁеҶ…йғЁзұ»еһӢзҡ„д»Јз ҒгҖӮ

з”ұдәҺеҶ…йғЁзұ»еһӢзҡ„ж ҮиҜҶз¬ҰдјҡжҸҗеҚҮеҲ°еӨ–йғЁзұ»еһӢдёӯпјҢжүҖд»ҘеҶ…йғЁзұ»еһӢе®һзҺ°зҡ„еӯ—ж®өпјҢж–№жі•е’ҢжҺҘеҸЈеңЁеӨ–йғЁзұ»еһӢдёӯд№ҹиғҪзӣҙжҺҘи®ҝй—®еҲ°гҖӮ

еҪ“еӨ–йғЁзұ»еһӢйңҖиҰҒе®һзҺ°дёҖдёӘе’ҢеҶ…йғЁзұ»еһӢдёҖж ·зҡ„ж–№жі•жҲ–жҺҘеҸЈж—¶пјҢеҸӘйңҖиҰҒз»ҷеӨ–йғЁзұ»еһӢйҮҚж–°з»‘е®ҡж–№жі•жҲ–е®һзҺ°жҺҘеҸЈеҚіеҸҜгҖӮ

**05.**зұ»еһӢе®һзҺ°жҺҘеҸЈ

GoиҜӯиЁҖдёӯпјҢжҺҘеҸЈжҳҜз”ЁжқҘе®ҡд№үиЎҢдёәзҡ„зұ»еһӢпјҢиҝҷдәӣиў«е®ҡд№үзҡ„иЎҢдёәдёҚз”ұжҺҘеҸЈзӣҙжҺҘе®һзҺ°пјҢиҖҢжҳҜйҖҡиҝҮж–№жі•з”ұз”ЁжҲ·е®ҡд№үзҡ„зұ»еһӢе®һзҺ°гҖӮ

еҰӮжһңз”ЁжҲ·е®ҡд№үзҡ„зұ»еһӢе®һзҺ°дәҶжҹҗдёӘжҺҘеҸЈйҮҢзҡ„дёҖз»„ж–№жі•пјҢйӮЈд№Ҳз”ЁжҲ·е®ҡд№үзҡ„иҝҷдёӘзұ»еһӢеҖјпјҢе°ұеҸҜд»ҘиөӢеҖјз»ҷиҜҘжҺҘеҸЈеҖјпјҢжӯӨж—¶з”ЁжҲ·е®ҡд№үзҡ„зұ»еһӢз§°дёәе®һдҪ“зұ»еһӢгҖӮ

иҖҢз”ЁжҲ·е®ҡд№үзҡ„зұ»еһӢжғіиҰҒе®һзҺ°дёҖдёӘжҺҘеҸЈпјҢйңҖиҰҒйҒөеҫӘдёҖдәӣи§„еҲҷпјҢиҝҷдәӣи§„еҲҷдҪҝз”Ёж–№жі•йӣҶжқҘиҝӣиЎҢе®ҡд№үгҖӮ

д»Һзұ»еһӢе®һзҺ°ж–№жі•зҡ„жҺҘж”¶иҖ…и§’еәҰжқҘзңӢпјҢеҸҜд»ҘжҸҸиҝ°дёәд»ҘдёӢиЎЁж јгҖӮ
